# 📊 DANA Sentiment & Topic Analysis Dashboard

## 📝 Project Overview
This project aims to analyze user sentiment of the DANA app on the Google Play Store and identify key topics frequently discussed through reviews. I used the IndoBERT model for sentiment classification and BERTopic for automatic topic modeling.

## 🚀 Key Features
- Sentiment Analysis: Classify review text into Positive and Negative using a Deep Learning model.
- Topic Modeling: Group thousands of reviews into specific topics (e.g., login, transactions, paylater) using BERTopic.
- Interactive Dashboard: Visualize data and test new sentences in real-time with Streamlit.

## 🛠️ Tech Stack
- Language: Python
- Libraries: Transformers (Hugging Face), PyTorch, BERTopic, Streamlit, Pandas, Scikit-learn, Google Play Scraper
- Deployment: Hugging Face (Model Weights) & Streamlit Cloud

## Process
- Scraping: Fetching real-world data from the Play Store.
- Cleaning: Tidying up messy Indonesian text.
- Modeling: Training IndoBERT and BERTopic (Deep Learning).
- Hugging Face: Storing models in the cloud.
- Deployment: Creating interactive applications with Streamlit.
