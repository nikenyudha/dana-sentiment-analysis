# 📊 Dana Sentiment & Topic Analysis Dashboard

## 📝 Project Overview
Proyek ini bertujuan untuk menganalisis sentimen pengguna aplikasi DANA di Google Play Store dan mengidentifikasi topik utama yang sering dibicarakan melalui review. Saya menggunakan model IndoBERT untuk klasifikasi sentimen dan BERTopic untuk pemodelan topik secara otomatis.

## 🚀 Key Features
Sentimen Analysis: Klasifikasi teks review menjadi Positif dan Negatif menggunakan model Deep Learning.

Topic Modeling: Mengelompokkan ribuan review menjadi topik-topik spesifik (misal: login, transaksi, paylater) menggunakan BERTopic.

Interactive Dashboard: Visualisasi data dan uji coba kalimat baru secara real-time dengan Streamlit.

## 🛠️ Tech Stack
Language: Python

Libraries: Transformers (Hugging Face), PyTorch, BERTopic, Streamlit, Pandas, Scikit-learn, google play scraper

Deployment: Hugging Face (Model Weights) & Streamlit Cloud
